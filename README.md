#3DLight
Open Source 3D Model and Animation Creation Tool

## Features
  - Import/Export files in *.obj, *.dae & *.3ds
  - Animation, keyframe & Bone
  - Default Models included
  - Sculping (soft & hard)
  - Vertex, Face & Point editing
  - Texturing

## You need
  - Library: WxWidgets
  - Compiler: MinGW
  - Recommended IDE's
    - Visual Studio 2013
    - Code::blocks
    - wxDev-C++
